# Markdown Reader Extension (Support)

Welcome to the support page for the **Markdown Reader** browser extension. 
This minimalist tool renders local and remote `.md` files directly in your web browser securely and instantly.

## Bug Reports & Feature Requests
If you encounter any issues while using the extension or have ideas for improvement:
1. Please open an issue in this repository using the [Issues tab](../../issues).
2. Clearly describe the problem, including the browser version you are using.

## Privacy & Security
This extension is built with a zero-dependencies, privacy-first philosophy. 
There is no telemetry, no tracking, and all rendering happens strictly locally on your machine via standard DOM manipulation mapped under a strict Content Security Policy.
